**A fully local SEO audit tool — 110 checks, AI recommendations, and PDF reports. No API keys. No cloud. Everything runs on your machine.**

---

## What It Does

SEO Auditor analyzes any webpage and produces a detailed audit report covering technical SEO, content quality, accessibility, AI search optimization (AEO), and local SEO (GEO). It uses a locally running AI model (via [Ollama](https://ollama.com/)) to generate plain-language recommendations — no subscriptions, no data leaving your machine.

You get a scored report, a phased action plan, and exports in PDF, HTML, CSV, and JSON — all from a single command or a browser-based UI.

## How It Works

1. You give it a URL
2. It fetches the page and runs 110 checks across 8 SEO categories
3. Your local AI model (Ollama) reads the results and writes recommendations
4. You get a scored report with a prioritized action plan

No login. No account. No monthly fee.

## Requirements

- [Node.js](https://nodejs.org/) v20 or later
- [Ollama](https://ollama.com/) installed and running locally
- A pulled Ollama model — recommended: `qwen3-coder:30b` or `mistral`

## Installation

**1. Clone the repository**

```bash
git clone https://github.com/LifeSciForge/seo-auditor.git
cd seo-auditor
```

**2. Install dependencies**

```bash
npm install
```

**3. Install and start Ollama**

Download from [ollama.com](https://ollama.com/), then pull a model:

```bash
ollama pull mistral
```

> For best results use `qwen3-coder:30b` — requires ~20GB RAM. For lighter machines, `mistral` (7B) works well.

## Usage

### Option A — Web UI (recommended for non-technical users)

```bash
npm run dev
```

Open your browser at `http://localhost:3000`, enter a URL, pick an industry, and click **Run Audit**.

### Option B — Command Line

```bash
# Basic audit with AI recommendations
npx ts-node src/index.ts audit --url https://yoursite.com

# Specify industry for weighted scoring
npx ts-node src/index.ts audit --url https://yoursite.com --industry ecommerce

# Skip AI for a fast offline check
npx ts-node src/index.ts audit --url https://yoursite.com --skip-ai

# Audit multiple URLs
npx ts-node src/index.ts audit --url https://site1.com --url https://site2.com

# Save JSON alongside PDF
npx ts-node src/index.ts audit --url https://yoursite.com --json
```

#### All CLI options

| Flag | Default | Description |
|---|---|---|
| `--url <url>` | required | URL(s) to audit |
| `--industry <type>` | `generic` | `generic` `ecommerce` `saas` `content` `local` `research` |
| `--model <model>` | `qwen3-coder:30b` | Ollama model to use |
| `--ollama-host <host>` | `http://localhost:11434` | Ollama server address |
| `--skip-ai` | false | Skip AI recommendations (faster) |
| `--output <path>` | auto | Custom PDF output path |
| `--check-external` | false | Also check external links for 404s |
| `--max-links <n>` | 30 | Max links to crawl |
| `--timeout <ms>` | 10000 | Per-request timeout |
| `--json` | false | Also export JSON report |
| `--verbose` | false | Show AI recommendations in terminal |

## What Gets Checked

### Technical SEO
HTTPS, redirect chains, canonical tags, robots meta, sitemap, structured data (JSON-LD), page speed signals, Core Web Vitals hints.

### On-Page SEO
Title tag, meta description, H1/H2 structure, keyword usage, Open Graph tags, image alt attributes, internal linking.

### Content Quality
Word count, content depth, reading level, duplicate content signals, publish/update dates, E-A-T signals.

### Link Profile
Internal link health, broken links (404s), external link quality, anchor text diversity, nofollow usage.

### Accessibility
ARIA landmarks, skip links, form labels, lang attribute, viewport meta, color contrast signals.

### Performance
Page size, image optimization, render-blocking resources, lazy loading, response time.

### AEO — AI Search Optimization (45 checks)
FAQ sections, direct answer paragraphs, question-format headings, featured snippet optimization, structured Q&A content, citation signals, factual density — optimized for Perplexity, ChatGPT, Google SGE.

### GEO — Local SEO (25 checks)
NAP consistency (Name, Address, Phone), LocalBusiness schema, Google Maps embed, click-to-call, local keywords, review signals, opening hours. Automatically skipped for non-local sites.

| Industry | Focus |
|---|---|

| `ecommerce` | Technical SEO, On-Page, Performance, Link Profile |
